Battlelog Mobile for Windows Phone

Battlefield 3 stats made easy!

Download this project as a .zip file Download this project as a tar.gz file

Description

Battlelog Mobile retrieves your real time Battlefield 3 soldier information from EA / DICE Battlelog server to your Windows Phone. You are able to track your progression in easy, intuitive manner with lots of images and charts. Get yours from Microsoft Marketplace, for free.

All three platforms are supported (PC, PS3, XBox).

Battlelog Mobile is a open source project it it has nothing to do with official EA / DICE Battlefield 3 or Battlelog products. Project page can be found right here.

Status

Version 1.7.0.0 is available for download from Marketplace and version 1.8.0.0 is under development. See the Issue Tracker if you want to know what is being developed for next release. You are also welcome to contact us via email or Battlelog.
